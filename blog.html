System Requirements for upgrading to Windows 11.


Blog no. : TB02 | June 30, 2021


According to Microsoft Blog post Windows 11 will be available as a free download for existing Windows users during the 2021 holiday season but you'll need to make sure it can actually run on your PC.



System Requirements

These are the basic requirements for officially installing Windows 11 on a PC. If your device does not meet these requirements, you may not be able to install Windows 11 on your device.

Processor: 1 gigahertz (GHz) or faster with 2 or more cores on a compatible 64-bit processor or System on a Chip (SoC)
RAM: 4 gigabytes (GB)
Storage: 64 GB or larger storage device
Note: See below under “More information on storage space to keep Windows 11 up-to-date” for more details.
System firmware: UEFI, Secure Boot capable
TPM: Trusted Platform Module (TPM) version 2.0
Graphics card: Compatible with DirectX 12 or later with WDDM 2.0 driver
Display: High definition (720p) display that is greater than 9” diagonally, 8 bits per color channel
Internet connection and Microsoft accounts: Windows 11 Home edition requires internet connectivity and a Microsoft account to complete device setup.



Now, if you are finding it difficult to know that your system fulfils all the minimum system requirements or not, then no need to worry you can use Microsoft's PC Health Check app.

What's new in Windows 11 OS?


Blog no. : TB01 | June 30, 2021


Six-Year after the release of Windows 10, OS Microsoft has now released Windows 11 OS. Windows 11 has all the power and security of Windows 10 with a redesigned and refreshed look. It also comes with new tools, sounds, and apps. It turns out Windows 10 wasn't the last version of Windows after all but worries, not Windows 11 will be free to upgrade on a Windows 10 PC or laptop.



KEY FEATURES

Here are some key features of the new Windows 11 OS which makes it different and better than Windows 10 OS:

New User Interface: The new Windows 11 user interface and Start menu look very similar to Windows 10X. Microsoft is using the concept of the rounded corner which gives it a refreshing and cool look.
The New Start menu is a clean and simplified version of what currently exists in Windows 10, Live Tiles are removed from the updated Start menu. It's been a while I am using Windows 10 and I personally never liked those live tiles. Updated Start menu also includes pinned apps, recent files, and quickly shut down or restart option in Windows 11 devices.
In Windows 11 Start button has been moved from the bottom left to the bottom centre of the screen and Unlike the previous windows version now Taskbar has been fixed to the bottom centre of the screen at least for now.


Microsoft Teams: Many of us use Microsoft Teams for online classes and meetings but now it will be built directly into the operating system, making it easier to access for daily use. You can connect with others on Teams across Windows, Android, Mac or iOS.


Android apps: One of the best features that will be coming to Windows 11 is Android Apps which will be installable from within the Microsoft Store using the [built-in] Amazon App Store. Microsoft will use Intel bridge technology to bring it to life so it's just seamless and smooth.


Snap Control: Windows 11 comes with the feature of snap controls that can be accessed from the maximize button on all apps which is ideal for users that wish to use multiple apps at once.
